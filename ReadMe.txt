Steps to setup the database locally:-

Step 1. Go to User accounts tab in phpMyAdmin.
Step 2. Add a user account named "artzteca" with the credentials in constants.php(Grant all permisions and database with same name.)
Step 3. Import the artzteca.sql in the database artzteca.
Step 4. Login with sample user accounts mentioned in account_details.txt
